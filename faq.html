Expense Tracker tracks our daily expenses like grocery payments, utility bills, online purchases datewise. User can monitor their daily expenses using this web app and also can download the summary for monthly budget analysis. Can filter expenses based on selected date criteria. Also can download Weekly and Monthly reports.

To add a fresh expense details, in home page there is a Add Expense button on top right corner. After clicking on it first select expense date→amount→description after that click on Add button. Now you can see the recently added expense is showing in the table. All newly added expense details will be displayed at last row of the table. If there are more than 10 items than the pagination will be displayed to go next section of records.

To filter the records we have to select the date from the date input filed, a pop-up calendar will be open than select the required date. After date selection click on the Filter button. According the user date selection, the expense data will be displayed.

Yes absolutely, user can download Daily / Weekly / Monthly expenses report. Just click on the added three buttons according to your choice, and a popup window will be displayed to save the document as pdf.
